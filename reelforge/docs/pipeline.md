# 制作管线 / 本地·云路由 / 反馈系统

## 1. 制作管线全景

MVP 覆盖到「选片 / 导出」，剪辑及之后由独立剪辑 Agent 后置。

| 阶段 | 内容 | MVP |
|---|---|---|
| 0. 前期 | 创意 → 剧本 → 镜头清单（谁/干什么/景别/运镜/时长/对白）。Agent 主场 | ✅ |
| 1. 资产 | 角色多视角定稿 + 表情姿态表 / 服装 / 道具 / 场景背景 / 美术风格 | ✅ |
| 2. 分镜 | 每镜头起始帧构图（角色+服装+道具摆进背景）+ 动线 + 需要时出尾帧 | ✅ |
| 3. 生成 | 图生视频（±尾帧）+ 运镜/动作控制 + 每镜头多 take | ✅ |
| 4. 选片 | 多 take 对比选优 + 补帧 / 超分 | ✅ |
| 5. 导出 | 导出片段 + 工程（给外部 NLE 收尾，B 方案） | ✅ |
| (后置) 后期 | 剪辑 / 音轨 / 口型同步 / 调色 / 字幕 / 和片 | ⏳ 独立剪辑 Agent |

贯穿全程的暗线：**一致性**（见 [agent-system.md](agent-system.md) §5）。

## 2. 本地 / 云路由

5090 本地为主，即梦 / Qwen 云补充：

| 阶段 | 主力 | 说明 |
|---|---|---|
| 定稿图 / 服装 / 道具 / 背景 / 风格 | **本地 5090** | Flux / Qwen-Image / SDXL + 一致性工具；可训 LoRA |
| 分镜关键帧合成 | **本地 5090** | inpaint / controlnet / 区域控制 |
| 图生视频镜头 | **本地 Wan 2.x ↔ 云即梦 / Qwen 路由** | 本地出草稿/可控；高质量或特定运镜走云。Agent 按镜头选 |
| 补帧 / 超分 | 本地 5090 | |
| 剪辑 / 调色 / 字幕 / 音轨 / 口型 | 后置（剪辑 Agent） | ComfyUI 不擅长，独立做 |

**智能路由**：导演 Agent 按「质量 / 速度 / 成本 / 可用性」选后端；允许用户固定偏好
（如「草稿走本地、定稿走即梦」）。**每次云调用先预估、再确认、累计账单**，避免烧钱惊吓。

云适配（异步作业模型 `submit → task_id → poll → 下载`）：
- **即梦 / Dreamina**：火山引擎（Volcengine）视觉/视频接口。
- **Qwen 通义万相（Wan）**：阿里云 DashScope / 百炼，文生/图生视频。
- 适配器统一管轮询、重试、限流、密钥、费用预估。

## 3. 反馈系统

ComfyUI 后端通过 WebSocket 推 `progress`（采样步数）、`executing`（当前节点）、
`executed`（节点产物，含预览图）等事件。Orchestrator 聚合后转发给 Mac，画布做叠加层：

- **节点级进度**：执行中节点高亮 + 进度条；完成打勾；失败标红并把报错翻成人话。
- **实时中间预览**：采样过程用 TAESD / latent 预览「正在长出来的画面」；出图 / 出帧节点贴缩略图。
- **管线总进度 + 预计剩余 + 预计花费**（云模型尤其重要）。
- **产物时间线 / 版本**：每次运行产出留档，可对比、回滚、「基于这次再改」。
- **断点续跑 / 局部重跑**：改后半段参数只重算受影响下游（ComfyUI 缓存天然支持）。
- **选片视图**：多 take 并排对比、打分，Agent 可做自动初筛（美学评分预筛）。

「尽量多反馈」是产品的明确诉求——进度、过程产物、预览三者要在画布上随时可见。
